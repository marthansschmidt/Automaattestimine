# 3.2 Seleniumiga Google.com testimine

See projekt kasutab Pythonit ja Seleniumi, et automatiseerida Google'i otsingut.

## Projekti eesmärk

Programmi eesmärk on avada Google'i avaleht, sisestada kasutaja antud otsingupäring ja kuvada esimesed otsingutulemused terminalis.

## Kasutatud tehnoloogiad

- Python
- Selenium
- Google Chrome
- Selenium WebDriver

## Paigaldamine

Python peab olema arvutisse installitud.

Seejärel tuleb installida Selenium:

```cmd
py -m pip install selenium
